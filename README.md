## ğŸ¯ About The Project

**Best RPG** is a **Java-based role-playing game engine** developed as part of a school curriculum.

The objective of this project was to design a **modular and extensible RPG architecture**, focusing on:

- Object-oriented programming principles
- Separation of concerns
- Game logic and turn-based combat flow
- Clear distinction between interfaces and implementations
- Centralized game state and combat resolution

Key elements of the project include:

- Turn-based combat system
- Character and role management
- Attack and damage handling logic
- Extensible architecture for future features

This project emphasizes **clean architecture and core Java concepts** rather than graphics or UI frameworks.

## âš”ï¸ Getting Started

This project runs locally using a Java and Maven environment.

### Prerequisites

- Java JDK (8 or higher)
- Maven
- A Java-compatible IDE (IntelliJ IDEA, Eclipse, or similar)

### Installation

1. Clone the repository:

       git clone https://github.com/joelle-jnbaptiste/SchoolProject---Best-RPG.git

2. Open the project in your IDE as a Maven project.

3. Ensure Maven dependencies are correctly resolved.

4. Run the `Main` class to start the RPG simulation.
